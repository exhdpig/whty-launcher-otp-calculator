# WHTY E-Schoolbag 管理员密码计算器 (OTP Generator)

本项目提供 WHTY 教育平板桌面 (`com.whty.eschoolbag.launcher`) 管理员验证码的离线计算方案。

> **⚠️ 警告**：经实测，系统更新后已删除该入口，此方法彻底作废。或许可以通过不联网方式先解锁，大家可以试试。

---

## 核心特性

* **全型号通用**：经逆向确认，该系统使用全局硬编码固定密钥。所有同型号、同版本 WHTY 平板的管理员密码计算逻辑完全一致。
* **动态更新**：验证码基于时间轴每 **30 分钟** 自动变换一次。
* **离线可用**：无需联网，只需同步系统时间即可计算。

---

## 使用指南

### 1. 准备工作
获取该型号平板对应的 **固定密钥 (Secret)**。
> **注**：由于改方法彻底作废，在此直接提供密钥为12345678901234567890。

### 2. 计算步骤
1. **同步时间**：确保平板系统时间准确（与标准北京时间一致）。
2. **执行运算**：将密钥与当前 30 分钟的时间片进行 HMAC-SHA1 运算。
3. **获取密码**：取运算结果的最后 6 位纯数字即可解锁。
